## 📋 Sobre o Projeto

O **Visualizador de Perfil do GitHub** permite que você digite o nome de usuário de qualquer pessoa no GitHub e visualize instantaneamente:
- Foto de perfil (Avatar).
- Bio e nome do usuário.
- Contadores de Seguidores e Seguindo.
- Lista dos 10 repositórios mais recentes, incluindo estatísticas de estrelas, forks e visualizações.

## ✨ Funcionalidades

- **Busca Dinâmica:** Pesquise por qualquer usuário ativo.
- **Atalhos de Teclado:** Suporte para busca pressionando a tecla `Enter`.
- **Interface Responsiva:** O design se adapta a diferentes tamanhos de tela (celulares, tablets e desktops).
- **Feedback Visual:** Exibe mensagens de "Carregando" e avisos caso o usuário não seja encontrado.
- **Link Direto:** Clique nos cartões de repositório para ser levado diretamente para o GitHub.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica do projeto.
- **CSS3:** Estilização com variáveis, flexbox e animações modernas.
- **JavaScript (ES6+):** Lógica de busca, consumo de API (`fetch`) e manipulação do DOM.
- **GitHub API:** Fonte de dados em tempo real.
- **DevIcons:** Ícones sociais e de tecnologias.

## 🚀 Como Executar o Projeto

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/seu-usuario/visualizador-perfil-github.git
    ```
2.  **Navegue até a pasta do projeto:**
    ```bash
    cd visualizador-perfil-github
    ```
3.  **Abra o arquivo `index.html`:**
    Basta abrir o arquivo diretamente no seu navegador ou usar a extensão *Live Server* do VS Code.
